# Java 编程测试题目

## 元数据
- **标题**: Java 编程测试题目集
- **作者**: 开发团队
- **日期**: 2026-04-06
- **版本**: 1.0.0

## 概述

本测试题目集旨在评估开发人员的Java编程能力、设计模式应用能力以及系统架构设计能力。所有题目均为控制台程序，不依赖任何UI框架，专注于核心业务逻辑和架构设计。

## 技术要求

- **Java版本**: Java 17+
- **构建工具**: Maven 或 Gradle（可选）
- **第三方依赖**: 不允许使用第三方框架（如Spring、Guice等）
- **代码规范**: 遵循阿里巴巴Java开发手册
- **测试要求**: 每个题目需编写单元测试，测试覆盖率不低于80%

## 题目列表

### [Task 1: 工作流引擎](./task1-workflow-engine/README.md)
实现一个可配置的工作流引擎，支持任务节点、条件分支、并行执行、状态持久化和恢复。

**难度**: ⭐⭐⭐⭐⭐  
**预计时间**: 8-12小时  
**核心技能**: 状态模式、策略模式、责任链模式

### [Task 2: 事件总线](./task2-event-bus/README.md)
实现一个发布-订阅模式的事件总线，支持同步和异步事件处理、事件过滤和路由、死信队列。

**难度**: ⭐⭐⭐⭐  
**预计时间**: 6-10小时  
**核心技能**: 观察者模式、生产者-消费者模式、线程池管理

### [Task 3: 依赖注入容器](./task3-dependency-injection/README.md)
实现一个轻量级依赖注入容器，支持构造器注入和字段注入、单例和原型作用域、循环依赖检测。

**难度**: ⭐⭐⭐⭐⭐  
**预计时间**: 10-15小时  
**核心技能**: 反射、依赖注入模式、图算法

## 提交要求

1. **代码结构**
   - 每个题目独立成一个子项目
   - 包含完整的源代码和测试代码
   - 提供清晰的README文档

2. **文档要求**
   - 设计思路说明

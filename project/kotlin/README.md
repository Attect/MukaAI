# Kotlin编程测试题目

## 概述

本目录包含一系列Kotlin编程测试题目，旨在考察数据结构与算法实现能力。所有题目均为纯控制台程序，无UI依赖。

## 题目列表

### Task 1: LRU缓存实现
**目录**: `task1-lru-cache/`
**难度**: 中等
**考察点**: 数据结构设计、并发编程、性能优化

### Task 2: 表达式求值器
**目录**: `task2-expression-evaluator/`
**难度**: 中等
**考察点**: 编译原理基础、递归下降解析、错误处理

### Task 3: 数据处理管道
**目录**: `task3-data-pipeline/`
**难度**: 较难
**考察点**: 函数式编程、设计模式、并行处理

## 技术要求

- **语言版本**: Kotlin 1.9+
- **构建工具**: Gradle或Maven
- **依赖限制**: 仅使用Kotlin标准库，不使用第三方库
- **代码风格**: 遵循Kotlin官方编码规范

## 验收标准

每个题目都包含详细的验收标准，包括：
- 功能测试用例
- 性能测试要求
- 代码质量要求
- 文档完整性要求

## 提交要求

1. 每个任务独立提交到对应目录
2. 必须包含完整的源代码和测试代码
3. 必须包含README.md说明文档
4. 代码必须通过所有测试用例

## 评分标准

- 功能完整性: 40%
- 代码质量: 30%
- 性能优化: 20%
- 文档完整性: 10%

## 注意事项

- 所有题目均需考虑边界情况和异常处理
- 代码必须具有良好的可读性和可维护性
- 测试覆盖率需达到80%以上
- 性能测试必须满足题目要求

## 开发环境设置

```bash
# 安装Kotlin编译器
sdk install kotlin

# 验证安装
kotlinc -version

# 创建项目结构
mkdir -p src/main/kotlin
mkdir -p src/test/kotlin
```

**技术栈**:
- Kotlin/JVM 2.3.10
- Ktor 3.4.1
- Kotlinx Serialization 1.6.2
- SecureRandom（安全随机数生成）

**关键发现**:
- 使用 `SecureRandom` 生成安全的随机 Token（32 字节，64 字符十六进制）
- 预设 Token 每日自动更换，降低泄露风险
- 专属 Token 永不过期，简化客户端管理
- Token 持久化使用 JSON 格式，易于管理和审计
- 支持 Token 撤销功能，增强安全性
- 多客户端支持良好，每个客户端独立 Token
